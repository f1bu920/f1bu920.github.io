Spring/SpringBootеёёз”ЁжіЁи§Ј

еҸ‘иЎЁдәҺ 2020-05-04 | еҲҶзұ»дәҺ SpringBoot

1. @SpringBootApplication

иҝҷдёӘжіЁи§ЈжҳҜSpringBootйЎ№зӣ®зҡ„еҹәзҹіпјҢжүҖжңүеҲӣе»әзҡ„SpringBootйЎ№зӣ®йғҪдјҡй»ҳи®ӨеңЁдё»зұ»дёҠеҠ дёҠиҝҷдёӘжіЁи§ЈгҖӮ

жҲ‘д»¬еҸҜд»ҘжҠҠ@SpringBootApplicationзңӢжҲҗжҳҜ@ConfigurationгҖҒ@EnableAutoConfigurationе’Ң@ComponentScanжіЁи§Јзҡ„йӣҶеҗҲгҖӮ

иҝҷдёүдёӘжіЁи§Јзҡ„дҪңз”ЁеҲҶеҲ«еҰӮдёӢпјҡ

  • @EnableAutoConfigurationпјҡеҗҜеҠЁSpringBootзҡ„иҮӘеҠЁй…ҚзҪ®жңәеҲ¶гҖӮ
  • @ComponentScanпјҡжү«жҸҸиў«@Component(@ServiceгҖҒ@Controller)жіЁи§Јзҡ„BeanпјҢжіЁи§Јдјҡй»ҳи®Өжү«жҸҸиҜҘзұ»жүҖеңЁеҢ…дёӢзҡ„жүҖжңүзұ»гҖӮ
  • @Configurationпјҡе…Ғи®ёеңЁSpringдёҠдёӢж–ҮдёӯжіЁеҶҢйўқеӨ–зҡ„BeanжҲ–еҜје…Ҙе…¶д»–й…ҚзҪ®зұ»гҖӮ

2. Spring Beanзӣёе…і

2.1 @Autowired

иҮӘеҠЁеҜје…ҘеҜ№иұЎеҲ°зұ»дёӯпјҢиў«жіЁе…Ҙиҝӣзҡ„зұ»еҗҢж ·иҰҒиў«Springе®№еҷЁз®ЎзҗҶгҖӮдҫӢеҰӮпјҡAdminUserServiceImplжіЁе…ҘеҲ°AdminControllerзұ»дёӯгҖӮ

2.2 @ComponentгҖҒ@RepositoryгҖҒ@ServiceгҖҒ@Controller

жҲ‘д»¬дёҖиҲ¬дҪҝз”Ё@AutowiredжіЁи§ЈжқҘи®©Springе®№еҷЁеё®жҲ‘д»¬иҮӘеҠЁиЈ…й…ҚBeanгҖӮжҲ‘д»¬еҸҜд»ҘдҪҝз”Ё@ComponentгҖҒ@RepositoryгҖҒ@Serviceе’Ң@ControllerжіЁи§Је°Ҷзұ»ж Үи®°дёәеҸҜз”ЁдәҺ@AutowiredиҮӘеҠЁиЈ…й…Қзҡ„зұ»гҖӮ

  • @ComponentпјҡйҖҡз”Ёзҡ„жіЁи§ЈпјҢеҸҜж Үи®°д»»ж„Ҹзұ»дёәSpringз»„д»¶гҖӮеҰӮжһңдёҖдёӘзұ»дёҚзҹҘйҒ“еұһдәҺйӮЈдёҖеұӮпјҢеҸҜд»ҘдҪҝз”ЁжӯӨжіЁи§ЈгҖӮ
  • @ReponsitoryпјҡеҜ№еә”жҢҒд№…еұӮеҚіDAOеұӮпјҢдё»иҰҒз”ЁдәҺж•°жҚ®еә“зӣёе…іж“ҚдҪңгҖӮ
  • @ServiceпјҡеҜ№еә”жңҚеҠЎеұӮгҖӮдё»иҰҒиҙҹиҙЈдёҖдәӣеӨҚжқӮзҡ„йҖ»иҫ‘пјҢйңҖиҰҒз”ЁеҲ°DAOеұӮгҖӮ
  • ControllerпјҡеҜ№еә”Spring MVCзҡ„жҺ§еҲ¶еұӮгҖӮдё»иҰҒз”ЁдәҺжҺҘеҸ—з”ЁжҲ·иҜ·жұӮе№¶и°ғз”ЁServiceеұӮиҝ”еӣһж•°жҚ®з»ҷеүҚз«ҜгҖӮ

2.3 @RestController

@RestControllerжіЁи§ЈжҳҜ@Controllerе’Ң@ResponseBodyжіЁи§Јзҡ„еҗҲйӣҶпјҢиЎЁзӨәиҝҷжҳҜдёӘжҺ§еҲ¶еҷЁBeanпјҢе№¶дё”е°Ҷиҝ”еӣһеҖјзӣҙжҺҘеЎ«е…ҘHTTPзҡ„е“Қеә”дҪ“дёӯпјҢжҳҜRESTйЈҺж јзҡ„жҺ§еҲ¶еҷЁгҖӮ

еҚ•зӢ¬дҪҝз”Ё@Controllerдё»иҰҒз”ЁдәҺиҝ”еӣһдёҖдёӘи§ҶеӣҫпјҢиҝҷжҳҜдј з»ҹSpring MVCзҡ„еә”з”ЁпјҢеұһдәҺеүҚеҗҺз«ҜдёҚеҲҶзҰ»зҡ„еңәжҷҜпјӣдҪҝз”ЁRestControllerжҲ–иҖ…дҪҝз”Ё@ControllerеҠ @ResponseBodyиҝ”еӣһJSONжҲ–XMLеҪўејҸзҡ„ж•°жҚ®пјҢз”ЁдәҺеүҚеҗҺз«ҜеҲҶзҰ»зҡ„еңәжҷҜгҖӮ

2.4 @Scope

еЈ°жҳҺSpring Beanзҡ„дҪңз”ЁеҹҹгҖӮеҰӮпјҡ@Scope("singleton")гҖӮе…¶жңүд»ҘдёӢдҪңз”Ёеҹҹпјҡ

  • singletonпјҡе”ҜдёҖBeanе®һдҫӢпјҢSpringдёӯзҡ„Beanй»ҳи®ӨйғҪжҳҜеҚ•дҫӢзҡ„гҖӮ
  • prototypeпјҡжҜҸж¬ЎиҜ·жұӮйғҪдјҡеҲӣе»әдёҖдёӘж–°зҡ„Beanе®һдҫӢгҖӮ
  • requestпјҡжҜҸж¬ЎHTTPиҜ·жұӮйғҪдјҡдә§з”ҹдёҖдёӘж–°зҡ„BeanпјҢиҜҘBeanд»…еңЁеҪ“еүҚHTTP requestеҶ…жңүж•ҲгҖӮ
  • sessionпјҡжҜҸж¬ЎHTTPиҜ·жұӮйғҪдјҡдә§з”ҹдёҖдёӘж–°зҡ„BeanпјҢиҜҘBeanд»…еңЁеҪ“еүҚHTTP sessionеҶ…жңүж•ҲгҖӮ

2.5 @Configuration

дёҖиҲ¬з”ЁдәҺеЈ°жҳҺй…ҚзҪ®зұ»пјҢеҸҜд»ҘдҪҝз”Ё@ComponentжіЁи§Јд»ЈжӣҝгҖӮ

3. еӨ„зҗҶеёёи§Ғзҡ„HTTPиҜ·жұӮзұ»еһӢ

еёёи§Ғзҡ„жңү5з§ҚиҜ·жұӮзұ»еһӢпјҡ

  • GETпјҡиҜ·жұӮд»ҺжңҚеҠЎеҷЁиҺ·еҸ–зү№е®ҡиө„жәҗгҖӮ
  • POSTпјҡеңЁжңҚеҠЎеҷЁдёҠеҲӣе»әдёҖдёӘж–°зҡ„иө„жәҗгҖӮ
  • PUTпјҡжӣҙж–°жңҚеҠЎеҷЁдёҠзҡ„иө„жәҗгҖӮе®ўжҲ·з«ҜжҸҗдҫӣжӣҙж–°еҗҺзҡ„ж•ҙдёӘиө„жәҗгҖӮ
  • DELETEпјҡеҲ йҷӨжңҚеҠЎеҷЁдёҠзҡ„зү№е®ҡиө„жәҗгҖӮ
  • PATCHпјҡжӣҙж”№жңҚеҠЎеҷЁдёҠдёҠзҡ„иө„жәҗпјҢе®ўжҲ·з«ҜжҸҗдҫӣжӣҙж”№зҡ„еұһжҖ§пјҢеҸҜзңӢжҲҗеұҖйғЁжӣҙж–°гҖӮ

3.1 GETиҜ·жұӮ

@GetMapping("/login")зӯүд»·дәҺ@RequestMapping(value="/login", method=RequestMethod.GET)гҖӮ

3.2 POSTиҜ·жұӮ

@PostMapping("/login")зӯүд»·дәҺ@RequestMapping(value="/login", method=RequestMethod.POST)гҖӮ

3.3 PUTиҜ·жұӮ

@PostMapping("/users/{userId}")зӯүд»·дәҺ@RequestMapping(value="/users/{userId}", method=RequestMethod.PUT)гҖӮ

3.4 DELETEиҜ·жұӮ

@DeleteMapping("/users/{userId}")зӯүд»·дәҺ@RequestMapping(value="/users/{userId}", method=RequestMethod.DELETE)гҖӮ

3.5 PATCH иҜ·жұӮ

@PatchMapping("/profile")зӯүд»·дәҺ@RequestMapping(value="/profile", method=RequestMethod.PATCH)гҖӮ

4. еүҚеҗҺз«Ҝдј еҖј

4.1 @PathVariableе’Ң@RequestParam

@PathValiableз”ЁдәҺиҺ·еҸ–и·Ҝеҫ„еҸӮж•°пјҢ@RequestParamз”ЁдәҺиҺ·еҸ–жҹҘиҜўеҸӮж•°гҖӮ

еҰӮжһңжҲ‘д»¬иҜ·жұӮзҡ„URLдёәпјҡ/blogs/edit/1пјҢиҺ·еҫ—зҡ„ж•°жҚ®е°ұжҳҜblogId=1гҖӮ

еҰӮжһңжҲ‘д»¬иҜ·жұӮзҡ„URLдёәпјҡ/blogs/save?blogTitle=titleпјҢиҺ·еҫ—зҡ„ж•°жҚ®е°ұжҳҜblogTitle=titleгҖӮ

4.2 @RequestBody

з”ЁдәҺиҜ»еҸ–RequestиҜ·жұӮзҡ„BodyйғЁеҲҶпјҢе№¶дё”Content-Typeдёәapplication/jsonж јејҸзҡ„ж•°жҚ®пјҢжҺҘж”¶еҲ°ж•°жҚ®еҗҺиҮӘеҠЁз»‘е®ҡеҲ°JavaеҜ№иұЎдёҠеҺ»гҖӮзі»з»ҹдјҡдҪҝз”ЁHttpMessageConverterжҲ–иҖ…иҮӘе®ҡд№үзҡ„HttpMessageConverterе°ҶиҜ·жұӮзҡ„bodyдёӯзҡ„jsonеӯ—з¬ҰдёІиҪ¬жҚўдёәJavaеҜ№иұЎгҖӮ

жіЁж„ҸпјҡдёҖдёӘиҜ·жұӮж–№жі•еҸӘеҸҜд»ҘжңүдёҖдёӘ@RequestBodyпјҢдҪҶеҸҜд»ҘжңүеӨҡдёӘRequestParamе’ҢPathVariableгҖӮ
