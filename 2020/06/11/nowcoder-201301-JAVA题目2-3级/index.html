思路

运用动态规划的思想，将右下角坐标看作(0,0)，左上角坐标看作(n,m)。从左上角移动到右下角等价于从右下角移动到左上角，且只能向左向上移动。令dp[i][j]代表从(0,0)移动到(i,j)的走法数目，则可以得到状态转移方程$dp[i][j] = dp[i][j-1]+dp[i-1][j],i>=1 AND j>=1$.

考虑初始化：当处于边界时，即i=0或j=0时，dp[i][j]都为1.
